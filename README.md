## Installation:
npm init

npm install mysql2

npm install sequelize

npm install dotenv


## Usage
Run the following commands at the root of your project.
1. mysql -u root -p
2. source db/schema.sql
3. quit <br>
After exiting the mysql prompts, run the following commands.
1. npm run seed
2. npm start
